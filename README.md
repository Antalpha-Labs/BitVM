# BitVM: Smarter Bitcoin Contracts

The official [BitVM](https://bitvm.org) implementation of [BitVM2](https://bitvm.org/bitvm2) running a [SNARK verifier](https://bitvm.org/snark). 

Work in progress.


## Overview 

- Winternitz signatures
- BigInt arithmetic
- Bn254
    - Field operations
    - Extension fields
    - Curve operations
    - Pairings
- Groth16 verifier
